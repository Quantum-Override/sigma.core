# sigma.core

A C project created with cproj.

## Building

This project uses the Bash Build System (BBS). See BUILDING.md for details.

## Directory Structure

- `src/`: Source files (.c)
- `include/`: Header files (.h)
- `test/`: Test files
- `build/`: Build artifacts
- `bin/`: Executables and libraries
- `logs/`: Test and build logs
- `docs/`: Documentation

## Usage

```bash
# Build the project
./cbuild

# Run tests
./ctest

# Clean build artifacts
./cbuild clean
```
